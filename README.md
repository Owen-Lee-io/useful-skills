## 简介

本仓库收集了一系列高质量的 Claude AI Skills，每个 Skill 都是一个独立的能力模块，为 AI 助手提供特定领域的专业知识和工作流程。这些 Skills 可以直接集成到 Claude 中使用，大幅提升工作效率。

## Skills 列表

### 📄 文档处理

#### **docx** - Word 文档处理
> 全面的 Word 文档创建、编辑和分析工具，支持修订追踪、评论、格式保留和文本提取。

**适用场景：**
- 创建新的 Word 文档
- 修改或编辑现有文档内容
- 处理修订追踪和评论
- 文档格式转换和分析

---

#### **pptx** - PowerPoint 演示文稿处理
> 演示文稿的创建、编辑和分析工具，支持布局管理、评论和演讲者备注。

**适用场景：**
- 创建新的演示文稿
- 修改幻灯片内容和布局
- 添加评论和演讲者备注
- 演示文稿结构分析

---

#### **xlsx** - Excel 表格处理
> 全面的电子表格创建、编辑和分析工具，支持公式、格式化、数据分析和可视化。

**适用场景：**
- 创建带公式和格式的新表格
- 读取或分析数据
- 修改表格同时保留公式
- 数据分析和可视化
- 公式重新计算

---

#### **pdf** - PDF 文档处理
> 全面的 PDF 处理工具包，支持文本和表格提取、PDF 创建、合并/拆分、表单填写等功能。

**适用场景：**
- 填写 PDF 表单
- 提取文本和表格数据
- 创建、合并或拆分 PDF
- 批量处理 PDF 文档

---

**适用场景：**
- 网站和落地页设计

### 📝 写作与沟通

#### **humanizer-zh** - 中文写作人性化
> 去除文本中的 AI 生成痕迹，使文字更自然、更像人类书写。基于维基百科的"AI 写作特征"综合指南。

**检测并修复：**
- 夸大的象征意义
- 宣传性语言
- 以 -ing 结尾的肤浅分析
- 模糊的归因
- 破折号过度使用
- 三段式法则
- AI 词汇滥用
- 否定式排比
- 过多的连接性短语

**适用场景：** 编辑或审阅文本，使其听起来更自然

---

## 使用方法

### 1. 克隆仓库

```bash
git clone https://github.com/Owen-Lee-io/useful-skills.git
cd useful-skills
```

### 2. 浏览 Skills

每个 Skill 都在 `skills/` 目录下有独立的文件夹，包含：
- `SKILL.md` - Skill 的主要定义和说明
- `LICENSE.txt` - 许可证信息（如适用）
- 其他支持文件（脚本、模板、字体等）

### 3. 集成到 Claude

根据你使用的 Claude 平台（API、Desktop、Web），将相应的 Skill 文件集成到你的环境中。具体方法请参考 Claude 官方文档。

### 4. 探索和定制

- 阅读每个 Skill 的 `SKILL.md` 文件了解详细用法
- 根据需要修改和定制 Skills
- 组合多个 Skills 以实现更复杂的工作流程

## 贡献指南

欢迎贡献新的 Skills 或改进现有 Skills！

### 贡献步骤：

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/AmazingSkill`)
3. 提交你的更改 (`git commit -m 'Add some AmazingSkill'`)
4. 推送到分支 (`git push origin feature/AmazingSkill`)
5. 开启一个 Pull Request

### Skill 规范：

每个 Skill 应包含：
- `SKILL.md` - 清晰的说明文档，包含 YAML 前置元数据
- 适当的许可证信息
- 示例和使用说明
- 必要的支持文件
