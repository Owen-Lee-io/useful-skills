# Flashcard Maker Skill

一个专门用于制作符合特定格式的 Flash 记忆卡片的 Claude Skill。

## 功能特性

✨ **自动生成 Flash 卡片** - 从学习材料自动提取知识点并生成标准格式的卡片  
🎨 **颜色高亮标记** - 使用特殊语法高亮显示关键信息  
📝 **挖空填空功能** - 创建填空型记忆点，增强主动回忆  
📚 **章节分组管理** - 自动按主题组织卡片  
🤖 **智能识别知识点** - AI 自动判断哪些内容需要高亮或挖空

## 安装方法

将 `flashcard-maker.skill` 文件放入以下任一目录：

- **个人全局使用**: `~/.claude/skills/`
- **项目专用**: `.claude/skills/`
- **VS Code**: `.github/skills/`
- **Cursor**: `.cursor/skills/`

## 使用示例

### 示例 1: 从文本生成卡片

**用户输入:**
```
帮我从这段文字生成记忆卡片：

"碳是一种化学元素，化学符号为 C，原子序数为 6。碳是非金属元素且为四价。碳的熔点是所有元素中最高的，为 3,823 K。"
```

**AI 输出:**
```
### 化学 - 元素

碳的化学符号是什么？
---
碳[T#!!fbc0bc#(C)]是原子序数为[F##6]的元素

碳的原子序数是多少？
---
碳的原子序数为[F##6]

碳的熔点是多少？
---
碳具有所有元素中最高的熔点，为[F##3,823 K]

碳是金属还是非金属？
---
碳是[F##非金属]元素且为[F##四价]
```

### 示例 2: 直接请求制作卡片

**用户输入:**
```
制作关于法国大革命的 Flash 卡片，包括开始时间、结束时间和主要成果
```

**AI 会自动:**
1. 生成相关知识点
2. 应用正确的格式
3. 添加适当的高亮和挖空标记

### 示例 3: 编辑现有卡片

**用户输入:**
```
帮我优化这张卡片的格式：
什么是光合作用？
光合作用是植物将光能转化为化学能的过程
```

## 卡片格式说明

### 基本结构
```
问题内容？
---
答案内容
```

### 特殊标记

**1. 颜色高亮** - `[T#!!fbc0bc#(内容)]`
```
氧[T#!!fbc0bc#(O)]是化学元素
```

**2. 挖空填空** - `[F##内容]`
```
碳的原子序数为[F##6]
```

### 章节分组
```
### 章节名称

卡片 1...
---
答案 1...

卡片 2...
---
答案 2...
```

## 技术细节

- **触发条件**: 用户提供学习材料 + 请求生成卡片，或明确说"制作 flashcard"/"生成记忆卡"
- **输出格式**: 标准化的三段式结构（问题 --- 答案）
- **标记语法**: 
  - 高亮颜色: `#fbc0bc` (浅粉/桃色)
  - 支持中英文混合内容

## 最佳实践

1. ✅ 每张卡片只包含一个核心概念
2. ✅ 问题清晰明确，不产生歧义
3. ✅ 答案简洁聚焦，不过于冗长
4. ✅ 战略性使用高亮 - 只突出真正关键的信息
5. ✅ 按逻辑主题分组卡片
6. ✅ 优先使用填空型问题增强主动回忆
